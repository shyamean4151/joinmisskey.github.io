  • : Search enabled
  • : reCAPTCHA/hCaptchaによるボット防御が有効

  • : Twitter integration available
  • : GitHub integration available
  • : Discord integration available
  • : 通知を常時配信可能 (iOSなど、利用環境によっては利用不可)

  • : インスタンスに登録されたユーザー数
  • : インスタンス内のユーザーのノートの総数
  • : Character limit
  • : 1ユーザーが利用できるドライブ容量
